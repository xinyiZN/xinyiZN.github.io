nextra：[官方文档](https://nextra.site/docs/guide/markdown)

```
# 使用Git Clone仓库到本地
git clone https://github.com/shuding/nextra-docs-template.git

# 进入项目目录
cd nextra-docs-template


# 安装依赖（确保已安装Node.js）
npm install

# 开发模式运行，自动打开浏览器
npm run dev

# 打包部署
npm run build
```

2. 在 next.config.js 里面配置

```
const withNextra = require("nextra")({
  theme: "nextra-theme-docs",
  themeConfig: "./theme.config.tsx",
  // 给文档中的代码块添加 copy 能力
  defaultShowCopyCode: true,
  // 支持 latex
  latex: true,
  // 支持静态图片
  staticImage: true,
})
const nextConfig = {
  output: "export",
  images: {
    unoptimized: true,
  },
}
module.exports = withNextra(nextConfig)

```

### 采用github action部署静态页面
注意：github生成的nextjs.yml里用的是/out，因此在next.config里需要配置：
```
const nextConfig = {
  output: "export",
  images: {
    unoptimized: true,//这里的配置是为了避免在page路由中next打包会报错的问题
  },
}
```
